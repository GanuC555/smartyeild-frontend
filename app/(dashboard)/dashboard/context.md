The dashboard is the main screen users see after logging in. It shows a live snapshot of their DeFi portfolio — how much they've deposited, how much yield they've earned so far, and what's available to spend.

- Shows principal balance, live yield counter (ticking every 100ms), and spendable balance
- Displays the two-pool split (liquid vs strategy) as a visual bar
- Shows strategy allocation breakdown as a donut chart
- Shows AI agent status cards (Guardian, Balancer, Hunter)
- Lists the 5 most recent transactions
- If no deposit yet, shows an onboarding CTA to the vault page
- Data refreshed every 30s via TanStack Query
